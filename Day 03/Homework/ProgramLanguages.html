ვებ დეველოპმენტი


ვებ დეველოპმენტი ნიშნავს საიტების გაკეთებას. აი, იცი რომ ტელეფონში ან კომპიუტერში რომ შედიხარ საიტებზე, ეგ არის ვებ დეველოპმენტის ნაშრომი.


საიტის აწყობა

საიტი ისე იშენება, როგორც სახლი. ჯერ ძირი უნდა დავდოთ (ეს არის HTML), მერე კედლები უნდა შევღებოთ და მოვაწესრიგოთ (ეს არის CSS), ბოლოს კი ელექტრონიკა და ყველაფერი რაც მოძრაობს და ფუნქციონირებს (ეს არის JavaScript).


რას ვიყენებთ ამისთვის?

HTML - ტექსტი და სტრუქტურა.

CSS - ფერები, ფორმები და დიზაინი.

JavaScript - ის, რაც საიტს ცოცხალივით ამუშავებს.
